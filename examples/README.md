# Mermaid Viewer Examples

This folder contains example Mermaid diagrams to help you get started with the Mermaid Viewer extension.

## Standalone `.mmd` Files

These files demonstrate different Mermaid diagram types in standalone `.mmd` format:

- **[flowchart.mmd](flowchart.mmd)** - Decision flowchart with styled nodes
- **[sequence.mmd](sequence.mmd)** - User authentication sequence diagram
- **[class-diagram.mmd](class-diagram.mmd)** - Object-oriented class hierarchy
- **[gantt.mmd](gantt.mmd)** - Project development timeline
- **[state-diagram.mmd](state-diagram.mmd)** - Application state transitions
- **[er-diagram.mmd](er-diagram.mmd)** - E-commerce database schema
- **[git-graph.mmd](git-graph.mmd)** - Git branching workflow
- **[pie-chart.mmd](pie-chart.mmd)** - Technology stack distribution

## Markdown File with Multiple Diagrams

- **[markdown-examples.md](markdown-examples.md)** - Shows how to embed multiple Mermaid diagrams in a Markdown document

## How to Use These Examples

1. **View in VS Code**: Open any `.mmd` file or `.md` file containing Mermaid code blocks
2. **Open Preview**: Click the preview icon in the editor title bar or use the command palette
3. **Explore Features**:
   - Change themes (Default, Dark, Forest, Neutral, Base)
   - Zoom in/out and pan around diagrams
   - Export diagrams as SVG, PNG, or JPG
   - Use CodeLens actions to preview individual diagrams

## Tips

- Use `Ctrl+Space` for autocomplete when writing Mermaid diagrams
- Syntax highlighting works in both `.mmd` files and Markdown code blocks
- Code folding is available for nested structures in `.mmd` files
